# Security Policy

## Supported Versions

Security fixes target the latest published version of `better-scrollbar`.

## Reporting a Vulnerability

Please do not open a public issue for suspected security vulnerabilities.

Report security concerns through GitHub private vulnerability reporting if it is
enabled for the repository. If private reporting is unavailable, contact the
maintainer through the email address listed on the maintainer's GitHub profile.

Include:

- Affected version or commit.
- Steps to reproduce the issue.
- Impact and any known workarounds.
- Whether the issue is already public.

The maintainer will acknowledge confirmed reports as soon as practical and will
coordinate disclosure timing for accepted vulnerabilities.
